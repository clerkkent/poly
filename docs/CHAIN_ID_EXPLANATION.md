# Chain ID 说明

## 什么是 Chain ID？

Chain ID（链 ID）是区块链网络的唯一标识符，用于区分不同的区块链网络。每个区块链网络都有一个唯一的数字 ID。

## 在 Polymarket 中的作用

Polymarket 支持多个区块链网络，不同的网络有不同的 Chain ID：

### 支持的 Chain ID

1. **137 - Polygon 主网（Polygon Mainnet）**
   - 这是 Polymarket 的主要生产网络
   - 真实的交易和资金
   - 需要真实的 MATIC 代币支付 Gas 费
   - 市场数据最丰富，流动性最好

2. **545 - 测试网（Testnet）**
   - 用于测试和开发
   - 使用测试代币，不涉及真实资金
   - 可以自由测试交易功能
   - 适合开发和调试

### 为什么需要 Chain ID？

1. **网络隔离**
   - 确保交易在正确的网络上执行
   - 防止在主网和测试网之间混淆

2. **API 路由**
   - Polymarket API 需要知道你要访问哪个网络的数据
   - 不同网络的市场数据是独立的

3. **交易签名**
   - 区块链交易签名需要包含 Chain ID
   - 防止重放攻击（一个网络的交易不能在另一个网络使用）

## 在系统中的应用

### 1. 账户配置

创建账户时，你可以指定 Chain ID：

```typescript
{
  name: "主账户",
  chainId: 137,  // Polygon 主网
  // 或
  chainId: 545,  // 测试网
}
```

### 2. 行情监控

你可以监控不同网络的市场：

```bash
# 监控 Polygon 主网
GET /api/markets?chainId=137

# 监控测试网
GET /api/markets?chainId=545
```

### 3. 交易执行

- 账户配置的 Chain ID 决定了交易在哪个网络执行
- 确保你的钱包连接到正确的网络
- 确保有足够的 Gas 代币（主网需要 MATIC，测试网需要测试代币）

## 如何选择 Chain ID？

### 使用 Polygon 主网 (137) 如果：
- ✅ 要进行真实交易
- ✅ 需要真实的流动性和市场数据
- ✅ 准备投入真实资金

### 使用测试网 (545) 如果：
- ✅ 只是测试和开发
- ✅ 不想使用真实资金
- ✅ 想尝试新功能

## 注意事项

1. **网络切换**
   - 切换 Chain ID 后，需要重新连接钱包
   - 确保钱包连接到对应的网络

2. **资金隔离**
   - 不同网络的钱包余额是独立的
   - 主网的 USDC 不能用于测试网

3. **市场数据**
   - 不同网络的市场是完全独立的
   - 主网的市场在测试网看不到

4. **Gas 费用**
   - 主网需要真实的 MATIC 支付 Gas
   - 测试网可以使用免费的测试代币

## 常见问题

**Q: 我可以同时监控多个网络吗？**
A: 可以！你可以在不同的浏览器标签页或使用不同的账户配置来监控不同网络。

**Q: 测试网的数据和主网一样吗？**
A: 不一样。测试网是独立的网络，市场数据、订单、余额都是独立的。

**Q: 如何获取测试网的代币？**
A: 通常通过测试网水龙头（faucet）获取，具体取决于测试网的类型。

**Q: Chain ID 可以修改吗？**
A: 可以，但修改后需要重新创建客户端连接。建议为不同网络创建不同的账户配置。
